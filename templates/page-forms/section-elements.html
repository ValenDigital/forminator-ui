Form Elements

Name Single Field

This field will allow yout to retrieve users' name with a single field.

Regular Markup 
<div class="forminator-col">

	<div class="forminator-field">

		<label for="field-name-single" class="forminator-label">Full Name</label>

		<input
			type="text"
			placeholder="E.g. John Doe"
			id="field-name-single"
			class="forminator-input"
		/>

		<span class="forminator-description" aria-describedby="field-name-single">
			Field description goes here.
		</span>

	</div>

</div>
Required Markup 
<div class="forminator-col">

	<div class="forminator-field forminator-is_required">

		<label for="field-name-single" class="forminator-label">
			Full Name
			<span class="forminator-required">*</span>
		</label>

		<input
			type="text"
			placeholder="E.g. John Doe"
			id="field-name-single"
			class="forminator-input"
			aria-required="true"
		/>

		<span class="forminator-error-message" aria-hidden="true">
			Required error message goes here.
		</span>

		<span class="forminator-description" aria-describedby="field-name-single">
			Field description goes here.
		</span>

	</div>

</div>
Name Multiple Field

This field will allow yout to retrieve users' name in a complex mode. This mode comes with different fields: prefix, first name, middle name, and last name.

Regular Markup 
<div class="forminator-col">

	<!-- Top row contains prefix and first name -->
	<div class="forminator-row">

		<div class="forminator-col">

			<div class="forminator-field">

				<label for="field-prefix" class="forminator-label">Prefix</label>

				<select id="field-prefix" class="forminator-select">
					<option>Mr.</option>
					<option>Mrs.</option>
					<option>Ms.</option>
					<option>Miss</option>
					<option>Dr.</option>
					<option>Prof.</option>
				</select>

				<span class="forminator-description">Optional description for prefix field.</span>

			</div>

		</div>

		<div class="forminator-col">

			<div class="forminator-field">

				<label for="field-first-name" class="forminator-label">First Name</label>

				<input
					type="text"
					placeholder="E.g. John"
					id="field-first-name"
					class="forminator-input"
				/>

				<span class="forminator-error-message" aria-hidden="true">This field is required.</span>

				<span class="forminator-description" aria-describedby="field-first-name">Optional description for first name field.</span>

			</div>

		</div>

	</div>

	<!-- Bottom row contains middle and last name -->
	<div class="forminator-row">

		<div class="forminator-col">

			<div class="forminator-field">

				<label for="field-middle-name" class="forminator-label">Middle Name</label>

					type="text"

			</div>

		</div>

		<div class="forminator-col">

			<div class="forminator-field">

				<label for="field-last-name" class="forminator-label">Last Name</label>
